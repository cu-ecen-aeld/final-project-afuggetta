# ECEN5713
### Final Project

Buildroot Configuration

The Buildroot configuration used for this project is stored in:

```
base_external/configs/buildroot_config
```

To reproduce the configuration:

```
cd buildroot
cp ../base_external/configs/buildroot_config .config
make olddefconfig
make
```
